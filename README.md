# YIVI 精酿手记 🍺

一个优雅的精酿啤酒品鉴记录应用，采用报刊风格设计。

## ✨ 功能特性

### 三大核心模块
- **醉境** - 品鉴日记：记录每一次与精酿的相遇
- **酒神教义** - 精酿指南：了解精酿啤酒的基础知识
- **酣战榜** - 名人堂：收藏你最爱的精酿

### 用户系统
- **用户模式**：完整的记录和管理功能
- **游客模式**：浏览查看，体验应用

### 技术特色
- 🎨 报刊风格设计，优雅简洁
- 📱 完全响应式，支持所有设备
- 🔐 数据本地存储，隐私安全
- 🎯 用户数据隔离，互不干扰
- 🌈 动态啤酒标签生成
- ⚡ 快速加载，流畅体验

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 在线部署

查看 [部署教程.md](./部署教程.md) 了解如何部署到Vercel或Netlify。

## 📦 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 下一代前端构建工具
- **Vue Router** - 官方路由管理器
- **Canvas API** - 动态标签生成
- **LocalStorage** - 本地数据存储

## 🎯 使用说明

1. **首次访问**：选择用户登录或游客访问
2. **用户登录**：输入任意用户名密码即可创建账号
3. **记录品鉴**：点击"记录新的一杯"添加啤酒记录
4. **查看统计**：在名人堂收藏你的最爱

## 📝 数据说明

- 所有数据保存在浏览器本地（LocalStorage）
- 每个用户的数据独立存储
- 游客模式使用公共数据，可以查看但不能修改
- 不会上传到服务器，完全隐私

## 🎨 设计灵感

灵感来自经典报刊排版：
- Libre Baskerville 衬线字体 - 优雅的标题
- Work Sans 无衬线字体 - 清晰的正文
- IBM Plex Mono 等宽字体 - 数据展示
- 温暖的纸张色调和墨水质感
