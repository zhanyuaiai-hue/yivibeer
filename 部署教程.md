# YIVI 精酿手记 - 部署到Vercel教程

## 🚀 快速部署步骤（10分钟完成）

### 第一步：准备工作

1. **确保项目有Git仓库**
   - 项目已经在 `D:\Git\mnt\d\YIVI\craft-beer-app`
   - 应该已经有Git管理

2. **注册Vercel账号**
   - 访问 https://vercel.com
   - 点击右上角 "Sign Up"
   - 推荐使用 GitHub 账号登录（最快）

---

### 第二步：推送代码到GitHub

打开命令提示符（CMD），执行以下命令：

```bash
# 1. 进入项目目录
cd D:\Git\mnt\d\YIVI\craft-beer-app

# 2. 检查是否已经初始化git
git status

# 如果没有git，初始化（如果已有则跳过此步骤）
git init
git add .
git commit -m "Initial commit - YIVI craft beer app"

# 3. 在GitHub创建新仓库
# 浏览器打开 https://github.com/new
# - 仓库名: yivi-craft-beer
# - 设置为 Public（公开）
# - 不要勾选任何初始化选项
# - 点击 "Create repository"

# 4. 关联GitHub仓库（替换YOUR_USERNAME为你的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/yivi-craft-beer.git
git branch -M main
git push -u origin main
```

---

### 第三步：部署到Vercel

#### 方法1：通过Vercel网站（推荐，最简单）

1. **登录Vercel**
   - 访问 https://vercel.com/dashboard
   - 点击 "Add New..." → "Project"

2. **导入GitHub仓库**
   - 点击 "Import Git Repository"
   - 授权Vercel访问你的GitHub
   - 选择 `yivi-craft-beer` 仓库
   - 点击 "Import"

3. **配置项目**
   - Framework Preset: 自动检测为 "Vite"
   - Build Command: `npm run build`（通常自动填写）
   - Output Directory: `dist`（通常自动填写）
   - 不需要添加环境变量
   - 点击 "Deploy"

4. **等待部署完成**
   - 通常需要1-3分钟
   - 完成后会显示 ✅ 和一个网址
   - 例如: `https://yivi-craft-beer.vercel.app`

#### 方法2：使用Vercel CLI（命令行）

```bash
# 1. 安装Vercel CLI
npm install -g vercel

# 2. 登录Vercel
vercel login

# 3. 部署项目
cd D:\Git\mnt\d\YIVI\craft-beer-app
vercel

# 按照提示操作：
# - 选择账号
# - 确认项目设置
# - 等待部署完成
```

---

### 第四步：获取你的网址

部署成功后，你会得到类似这样的网址：
- **Production**: `https://yivi-craft-beer.vercel.app`

这个网址可以：
- ✅ 在任何地方访问（不需要公司内网）
- ✅ 在手机、电脑、平板上打开
- ✅ 分享给朋友使用
- ✅ 自动HTTPS加密
- ✅ 全球CDN加速

---

### 第五步：更新部署

以后每次修改代码，只需：

```bash
# 1. 提交修改
git add .
git commit -m "更新说明"
git push

# 2. Vercel会自动检测并重新部署
# 通常1-2分钟后更新就会生效
```

## 🌐 其他免费部署选项

### Netlify（另一个优秀选择）

1. 访问 https://netlify.com
2. 注册账号（可用GitHub登录）
3. 点击 "Add new site" → "Import an existing project"
4. 选择GitHub仓库
5. Build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
6. 点击 "Deploy"

---

## ⚙️ 自定义域名（可选）

如果你有自己的域名：

1. 在Vercel项目设置中找到 "Domains"
2. 添加你的域名
3. 按照提示配置DNS
4. 等待生效（通常几分钟到几小时）

## 🔐 数据说明

- 所有数据保存在**浏览器本地**（localStorage）
- 不会上传到服务器
- 每个用户的数据独立存储
- 如果清除浏览器数据，记录会丢失
- 建议定期截图或导出备份
